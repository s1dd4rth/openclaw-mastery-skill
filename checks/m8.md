# Module 8 — Let It Write (5 deterministic + 1 judgment + 1 attestation)

Run each check in order. Always include manual checks with `pass: null, manual: true`. Check 2 is platform-conditional. Check 5 (`test-email-sent`) calls the `imap-smtp-email` skill — see notes below.

> CLI assumptions: `openclaw config get` for env-file inspection (without displaying values), `imap-smtp-email` skill exposes a Sent-folder scan action that the validator can invoke. Verify before tagging.

## 1. `smtp-configured`

Confirm SMTP keys are present in `~/.config/imap-smtp-email/.env` (key presence only — never display values).

```
grep -cE '^(SMTP_HOST|SMTP_PORT|SMTP_USER|SMTP_PASSWORD|SMTP_FROM)=.+' ~/.config/imap-smtp-email/.env
```

- **pass:** at least 4 SMTP_ keys are present with non-empty values
- **fail:** fewer than 4 SMTP_ keys present
- **detail (pass):** `"SMTP keys present (<N>/5 set)"`
- **detail (fail):** `"SMTP not configured (<N>/5 keys present)"`
- **evidence:** `{ "smtp_keys_present": <int>, "expected_min": 4 }`
- **fix_prompt:** `"Tell the Claw: 'Add my Gmail SMTP settings to ~/.config/imap-smtp-email/.env using my App Password.'"`

> Count keys only. Never include values in evidence.

## 2. `config-permissions` *(platform-conditional)*

Verify `~/.config/imap-smtp-email/.env` is still owner-only after Module 8's edits.

Linux: `stat -c "%a" ~/.config/imap-smtp-email/.env`
macOS: `stat -f "%A" ~/.config/imap-smtp-email/.env`

- **pass:** output equals `600`
- **fail:** anything else
- **detail (pass):** `"Permissions: 600 (owner-only)"`
- **detail (fail):** `"Permissions: <octal>; expected 600"`
- **evidence:** `{ "path": "<path>", "mode": "<octal>", "platform": "linux|macos" }`
- **fix_prompt:** `"Tell the Claw: 'Verify ~/.config/imap-smtp-email/.env is still permissions 600.'"`

## 3. `outbound-rules`

Confirm AGENTS.md contains "Outbound Email Protocols" with approval-gate keywords.

```
grep -iE 'outbound email protocols|show (the )?full draft.*before send|wait for.*approval' ~/.openclaw/workspaces/main/AGENTS.md
```

- **pass:** at least one match for both protocol and approval-gate language
- **fail:** missing
- **detail (pass):** `"AGENTS.md includes outbound email approval gate"`
- **detail (fail):** `"AGENTS.md missing outbound email protocols or approval rule"`
- **evidence:** `{ "matched_lines": <count> }`
- **fix_prompt:** `"Tell the Claw: 'Add Outbound Email Protocols to AGENTS.md: always show the full draft and wait for explicit approval before sending.'"`

## 4. `follow-up-exists`

Confirm `follow-up-email` workspace skill exists (existence only — content quality is check #6, manual).

```
test -f ~/.openclaw/workspaces/main/skills/follow-up-email/SKILL.md
```

- **pass:** file exists
- **fail:** missing
- **detail (pass):** `"follow-up-email workspace skill present"`
- **detail (fail):** `"follow-up-email workspace skill missing"`
- **evidence:** `{ "skill_path": "<path>", "exists": <bool> }`
- **fix_prompt:** `"Tell the Claw: 'Create the follow-up-email skill in the workspace skills folder now.'"`

## 5. `test-email-sent`

Scan the user's Gmail Sent folder via the `imap-smtp-email` skill for a self-addressed test email in the last 24 hours.

This check delegates to another skill. Invoke `imap-smtp-email`'s sent-folder scan action with parameters: `since: <24h ago>`, `to: <SMTP_FROM value>`, `headers_only: true`. Do NOT include message bodies in evidence.

- **pass:** at least one matching message found in the last 24h
- **fail:** none found
- **detail (pass):** `"Found test email in Sent folder, dated <timestamp>"`
- **detail (fail):** `"No self-addressed test email in Sent folder (last 24h)"`
- **evidence:** `{ "matches_found": <int>, "scan_window_hours": 24 }`
- **fix_prompt:** `"Tell the Claw: 'Verify SMTP_FROM matches my Gmail address, then send a test email to myself and confirm it appears in Sent.'"`

> If `imap-smtp-email` cannot be invoked from another skill (architectural limitation), return `pass: null, manual: true` with `detail: "Manual: imap-smtp-email cannot be called from validator on this OpenClaw version — user checks Sent folder by hand"`.
